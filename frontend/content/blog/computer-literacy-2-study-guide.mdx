## 결론 먼저

컴활 2급 필기는 **2주 × 하루 1–2시간**이면 비전공자도 합격 가능해요. 관건은 **스프레드시트 함수 6개(VLOOKUP, IF, IFS, COUNTIF, SUMIF, SWITCH)** 를 확실히 잡는 것. 아래에서 과목별 빈출 토픽·2주 일자별 플랜·자주 하는 실수를 정리합니다.

## 컴활 2급 필기, 얼마나 쉬운가

"컴활 2급은 그냥 가면 붙어요"라는 말이 있는데, 절반만 맞는 얘기예요. 필기 합격률이 60–70%대이긴 하지만, 준비 없이 들어가면 스프레드시트 과목에서 생각보다 많이 틀립니다.

다만 확실한 건, **2주면 충분하다**는 거예요. 1급처럼 3과목이 아니라 2과목이고, 문항도 40개라 범위가 좁습니다.

| 항목 | 내용 |
|------|------|
| 과목 | 컴퓨터 일반(20) + 스프레드시트 일반(20) |
| 문항 수 | 40문항 |
| 시간 | 40분 |
| 합격 | 평균 60점 + 과목별 40점 이상 |

1급과 비교하면 DB 과목이 없고, 시간도 40분이라 체감 부담이 확 줄어요.

---

## 1과목: 컴퓨터 일반

20문항 중 절반 이상이 **용어 암기**예요. 기출에서 반복되는 키워드만 잡으면 14–16개는 맞출 수 있습니다.

**빈출 토픽:**
- Windows 기능 (제어판, 파일 관리, 단축키) — 매회 3–4문제
- 네트워크/인터넷 (프로토콜, IP, DNS) — 매회 3–4문제
- 하드웨어 (CPU, 메모리, 저장장치) — 매회 3–4문제
- 정보 보안 (바이러스, 방화벽) — 2–3문제
- 멀티미디어, ICT 신기술 — 각 1–2문제

**공부법:**
기출 3회분을 먼저 풀어보세요. 모르는 용어를 따로 정리하고 그것만 외우면 됩니다. 교재를 처음부터 읽을 필요 없어요.

---

## 2과목: 스프레드시트 일반

여기가 **진짜 승부처**예요. 컴퓨터 일반은 대부분 맞추는데, 스프레드시트에서 갈리거든요. 오답률이 40–60%대라는 통계도 있습니다.

**빈출 토픽:**
- 함수 (SUM, IF, IFS, VLOOKUP, COUNTIF) — 매회 4–5문제
- 셀 참조 + 찾기/참조 함수 (SWITCH, 절대참조) — 2–3문제
- 데이터 관리 (정렬, 고급 필터, 부분합) — 2–3문제
- 차트 (종류, 속성, 보조 축) — 2–3문제
- 셀 서식, 조건부 서식 — 2–3문제
- 매크로/VBA, 피벗, 인쇄 — 각 1문제

**공부법:**
함수 10개만 확실히 외우면 절반은 해결돼요. VLOOKUP, IF, IFS, COUNTIF, SUMIF, SWITCH — 이 6개가 핵심이고, 나머지는 기출에서 나오는 패턴만 익히면 됩니다.

<Highlight color="amber">2024년 개편으로 IFS, SWITCH 함수가 추가됐어요. 이전 기출에는 안 나오니 별도로 챙겨야 합니다.</Highlight>

## 2주 학습 플랜

하루 1–2시간 기준:

| 기간 | 할 일 |
|------|------|
| **1–2일** | 기출 1회분 풀기. 과목별 약점 파악 |
| **3–5일** | 1과목 기출 용어 정리 + 암기 |
| **6–9일** | 2과목 함수 10개 암기 + 기출 함수 문제 집중 |
| **10–12일** | 기출 2–3회 연속 풀기. 과목별 점수 체크 |
| **13–14일** | 약한 부분 보강 + 오답 복습 |

> <Highlight color="green">핵심: 컴활 2급 필기는 기출 재활용률이 높은 시험이에요. 기출을 많이 풀수록 유리합니다.</Highlight>

## 자주 하는 실수

- **"2급이니까 그냥 가도 돼"** — 스프레드시트 과목 오답률이 40–60%라 무준비 탈락 흔함
- **함수를 처음부터 다 외우려 함** — 빈출 6개만 확실히 하고 나머지는 기출에서 익히는 게 효율적
- **2024 개편 전 기출만 풀기** — IFS·SWITCH가 빠져있어 실제 시험과 괴리
- **과목별 40점 과락 방어 안 챙김** — 평균 60점 넘겨도 한 과목이 39점이면 탈락

## 시험장에서의 접근법

- **40문항 × 40분 = 문제당 1분** — 아는 문제부터 먼저
- 1과목(컴퓨터 일반) 용어 문제는 10–15초 내 결정
- 함수 문제는 보기에 수식 결과가 있으니 매칭 방식
- 마지막 5분은 **과목별 점수 추정**으로 과락 위험 과목 재검토

## 1급과 뭐가 다른가

| 항목 | 1급 | 2급 |
|------|-----|-----|
| 과목 수 | 3과목 | **2과목** (DB 없음) |
| 문항 수 | 60문항 | **40문항** |
| 시간 | 60분 | **40분** |
| 난이도 | 중상 | **중하** |

DB 과목이 빠진 게 가장 큰 차이예요. SQL이나 Access를 몰라도 되니까 비전공자에게 훨씬 접근성이 좋습니다.
