## 1과목: 컴퓨터 일반

### 컴퓨터 세대

| 세대 | 주요 소자 | 특징 |
|------|---------|------|
| 1세대 | 진공관 | 발열 심함, 크기 거대 |
| 2세대 | 트랜지스터 | 크기 축소, 신뢰성 향상 |
| 3세대 | IC(집적회로) | 소형화, 운영체제 등장 |
| 4세대 | LSI/VLSI | 개인용 컴퓨터 등장 |

### 데이터 단위

비트(1) → 니블(4) → 바이트(8) → 워드(16/32/64)

### CPU 명령어 처리

**인출(Fetch) → 해독(Decode) → 실행(Execute) → 저장(Store)**

### 메모리 계층

레지스터 → 캐시 → RAM(주기억) → SSD/HDD(보조기억)

- **RAM**: 휘발성, 전원 꺼지면 데이터 소멸
- **ROM**: 비휘발성, BIOS 저장
- **캐시**: CPU와 RAM 속도 차이 보완. SRAM 사용

### Windows 단축키

| 단축키 | 기능 |
|--------|------|
| Shift + Delete | 휴지통 거치지 않고 영구 삭제 |
| Ctrl + Shift + Esc | 작업 관리자 |
| Alt + F4 | 프로그램 종료 |
| Windows + E | 파일 탐색기 |

### 네트워크

- **TCP**: 연결형, 신뢰성 보장, 3-way handshake
- **UDP**: 비연결형, 빠르지만 신뢰성 없음
- **HTTP**: 웹 페이지 전송 (포트 80)
- **FTP**: 파일 전송 (포트 21)
- **SMTP**: 메일 발송
- **DNS**: 도메인 → IP 변환

### OSI 7계층

응용(7) → 표현(6) → 세션(5) → 전송(4) → 네트워크(3) → 데이터링크(2) → 물리(1)

외우는 법: "**응표세전네데물**"

| 계층 | 전송 단위 |
|------|---------|
| 전송 계층 | 세그먼트 |
| 네트워크 계층 | 패킷 |
| 데이터링크 계층 | 프레임 |
| 물리 계층 | 비트 |

### 정보 보안

- **방화벽**: 외부 → 내부 접근 제어
- **IDS**: 침입 탐지 (감지만)
- **IPS**: 침입 방지 (감지 + 차단)
- **대칭키 암호화**: AES, DES (같은 키로 암호화/복호화)
- **비대칭키 암호화**: RSA (공개키 + 개인키)

## 2과목: 스프레드시트 일반

### 핵심 함수

| 함수 | 기능 | 예시 |
|------|------|------|
| **SUM** | 합계 | `=SUM(A1:A10)` |
| **IF** | 조건 분기 | `=IF(A1>=60, "합격", "불합격")` |
| **IFS** | 다중 조건 (2024 추가) | `=IFS(A1>90,"수", A1>80,"우", TRUE,"미")` |
| **VLOOKUP** | 세로 방향 찾기 | `=VLOOKUP(찾을값, 범위, 열번호, 0)` |
| **COUNTIF** | 조건부 개수 | `=COUNTIF(A1:A10, ">60")` |
| **SUMIF** | 조건부 합계 | `=SUMIF(조건범위, 조건, 합계범위)` |
| **SWITCH** | 값 매칭 (2024 추가) | `=SWITCH(A1, 1,"월", 2,"화", "기타")` |
| **LEFT/MID/RIGHT** | 문자열 추출 | `=LEFT("컴활",1)` → "컴" |
| **ROUND** | 반올림 | `=ROUND(3.456, 2)` → 3.46 |

<Highlight color="red">IFS와 SWITCH는 2024년 개편에서 추가된 함수예요. 이전 기출에는 안 나오니 별도로 공부해야 합니다.</Highlight>

### 셀 참조

- **상대참조** `A1`: 복사 시 자동 변경
- **절대참조** `$A$1`: 복사해도 고정
- **혼합참조** `$A1`: 열 고정, 행 변동 / `A$1`: 행 고정, 열 변동

### 표시 형식 코드

| 코드 | 의미 | 예시 |
|------|------|------|
| `#` | 유효 숫자만 표시 | `###` → 123 |
| `0` | 자리수 없으면 0 표시 | `000` → 023 |
| `#,##0` | 천 단위 구분 | 1,234 |
| `#,##0.00` | 소수 2자리 | 1,234.50 |

### 정렬 순서 (오름차순)

숫자 → 특수문자 → 영문(대→소) → 한글(가나다)

### 차트 종류

| 차트 | 용도 |
|------|------|
| 원형 | 전체 대비 비율 |
| 꺾은선 | 시간에 따른 추이 |
| 막대 | 항목 간 비교 |
| 분산형 | 두 변수 간 관계 |

### 피벗 테이블

- 대량 데이터를 **요약/분석**하는 대화형 테이블
- 4개 영역: **행, 열, 값, 필터**
- 원본 데이터는 변경되지 않음

### 매크로

- **매크로**: 반복 작업 자동화. `.xlsm`으로 저장
- **VBA**: 매크로의 프로그래밍 언어
- Sub 프로시저: 값 반환 안 함
- Function 프로시저: 값 반환
