## ADsP, 딸까 말까부터 정하고 시작하세요

ADsP는 **데이터 분석 입문자용 국가공인 자격증**이에요. 한국데이터산업진흥원이 주관하고, 상위 자격증인 ADP(데이터분석 전문가)의 디딤돌 역할을 합니다.

딸지 말지는 이렇게 판단하시면 됩니다.

- **데이터 직무 취업 준비 중** → 이력서 한 줄용으로 <Highlight color="green">추천</Highlight>
- **SQL/DB는 이미 어느 정도 아는데 분석 쪽도 보고 싶다** → <Highlight color="green">추천</Highlight> (SQLD와 중복 과목 거의 없음)
- **통계·R 아예 처음** → 부담 느낄 수 있지만, 2주 이상 투자 가능하면 <Highlight color="amber">OK</Highlight>
- **개발 직무 쪽만 준비** → 우선순위는 낮아요. SQLD·정처기가 더 가성비 좋음

## 1과목: 데이터 이해 (10문항)

이론·개념 위주라 **암기 시험**에 가깝습니다. 출제 포인트가 회차별로 비슷해요.

**거의 매회 나오는 것:**
- DIKW 피라미드 (데이터–정보–지식–지혜) — 사례 매칭 문제로 자주 출제
- 데이터베이스 특성 4가지 (통합·저장·공유·운영)
- 빅데이터 3V / 5V
- OLTP vs OLAP 차이
- 데이터 사이언티스트 Hard Skill / Soft Skill 구분

**공부법:**
교재 1회독 후 **기출 보기 표현에 익숙해지기**가 전부예요. "가장 옳은 것" / "가장 옳지 않은 것" 패턴이라, 애매한 보기 하나를 걸러낼 수 있으면 충분합니다.

---

## 2과목: 데이터 분석 기획 (10문항)

**분석 프로젝트의 절차/방법론**을 묻는 과목이에요. 실무 경험 있는 분들은 쉽게 느끼지만, 처음이면 용어 자체가 낯설어요.

**핵심 출제 포인트:**
- 분석 방법론 3가지: **KDD / CRISP-DM / SEMMA** 단계와 차이
- 분석과제 발굴: **하향식(Top-Down) vs 상향식(Bottom-Up)**
- 분석 성숙도 모델 (도입–활용–확산–최적화)
- 분석 거버넌스 4대 구성 요소 (조직·프로세스·시스템·데이터)

**공부법:**
방법론 이름이 많아서 헷갈리는데, <Highlight color="green">**단계 이름을 순서대로 외우는 것**</Highlight>이 제일 효율적이에요. CRISP-DM 6단계는 시험에 거의 매번 나와요. "Business → Data → Prep → Model → Eval → Deploy" 한 번에 외우세요.

## 3과목: 데이터 분석 (30문항)

**시험의 본체**예요. 여기서 점수를 확보하지 못하면 합격이 어렵습니다.

### 출제 빈도 높은 주제

<StatBar label="3과목 토픽별 출제 빈도 (체감)" unit="문제" color="#14b8a6" items="회귀분석:3.5:중상,분류분석:3.5:중상,통계 기초:3.5:중,R 기초:3.5:중,앙상블·신경망:2.5:상,연관분석:2.5:계산 필수,군집분석:2:중,모형 평가:1.5:중,시계열 분석:1.5:상,PCA·MDS:1:상" />

| 주제 | 체감 출제 수 | 난이도 |
|------|------------|--------|
| 회귀분석 (단순/다중/로지스틱/릿지/라쏘) | 3–4문제 | 중상 |
| 분류분석 (의사결정나무·SVM·KNN) | 3–4문제 | 중상 |
| 통계 기초 (가설검정·신뢰구간) | 3–4문제 | 중 |
| R 프로그래밍 기초 | 3–4문제 | 중 |
| 앙상블 (배깅·부스팅·랜덤포레스트) | 2–3문제 | 상 |
| 연관분석 (지지도·신뢰도·향상도 **계산**) | 2–3문제 | **계산 필수** |
| 군집분석 (K-means·계층적) | 2문제 | 중 |
| 모형 평가 (혼동행렬·ROC/AUC) | 1–2문제 | 중 |
| 시계열 분석 (ARIMA·정상성) | 1–2문제 | 상 |
| PCA / MDS | 1문제 | 상 |

### 계산 문제 반드시 연습하세요

<Highlight color="red">개편 이후 계산 문제 비중이 확실히 늘었어요.</Highlight>

- 연관분석: 지지도 = P(A∩B), 신뢰도 = P(B|A), 향상도 = 신뢰도 / P(B) — **직접 종이에 써보면서** 연습
- 혼동행렬: 정확도 / 정밀도 / 재현율 / F1 공식 구분
- 회귀 계수 해석과 결정계수(R²)

공식만 외우고 시험장에서 숫자 대입하려면 당황합니다. 기출 5문제 정도는 직접 풀어보세요.

### 릿지 vs 라쏘 (매회 나옵니다)

| 구분 | Ridge | Lasso |
|------|-------|-------|
| 규제 | **L2** (제곱) | **L1** (절대값) |
| 변수 선택 효과 | X | **O** (계수를 0으로) |

L1/L2 방향 헷갈리는 분들이 정말 많아요. "**라쏘 = L1 = 제거**" 한 세트로 외우세요.

## 2주 플랜 (전공자·SQL 경험자)

| 기간 | 할 일 |
|------|------|
| **1–2일차** | 1·2과목 교재 1회독 + 방법론 단계 암기 |
| **3–7일차** | 3과목 알고리즘별 정리 (회귀·분류·앙상블·군집·연관) |
| **8–11일차** | 기출/모의고사 3회 풀기. 틀린 문제 주제 기록 |
| **12–13일차** | 약한 주제 보강 + 계산 문제 집중 연습 |
| **시험 전날** | 오답 노트만 훑기. 새 개념 금지 |

## 1달 플랜 (비전공자·통계 처음)

| 주차 | 할 일 |
|------|------|
| **1주차** | 1·2과목 개념 잡기. 용어 정리 우선 |
| **2주차** | 3과목 전반부 (R·통계·회귀·분류) |
| **3주차** | 3과목 후반부 (앙상블·군집·연관·시계열·평가) + 기출 2회 |
| **4주차** | 모의고사 반복 + 계산 문제 집중 + 오답 정리 |

<Highlight color="amber">통계가 처음이면 2주는 무리예요. 무리하지 말고 1달 확보하는 게 정신건강에 좋습니다.</Highlight>

## CBT 활용법

ADsP도 개편 이후 **CBT(컴퓨터 시험)** 로 진행돼요. 실제 시험 환경은 모니터로 읽고 마우스로 체크하는 형태라, 종이 기출만 풀면 감각이 안 맞습니다.

- 모의고사를 **실제 시험 시간(90분)** 에 맞춰 풀어보세요
- 긴 지문 문제에서 **스크롤 습관** 들이기
- 헷갈리는 문제는 **표시 기능** 써서 마지막에 돌아보기

## 시험 당일 팁

- 1·2과목은 **30분 안에** 끝내고 3과목에 시간 몰빵
- 계산 문제는 **천천히, 숫자 두 번 확인**
- 모르는 문제는 일단 마킹하고 넘기기
- 릿지/라쏘, KDD/CRISP-DM 같은 비교 문제는 **표를 머릿속에 그리면서** 풀기

## 정리

ADsP는 과목별 성격이 다릅니다. 1·2과목은 암기, 3과목은 개념 + 계산이에요. <Highlight color="green">3과목이 30문항 중 절반에 가까운 14문항이라, 전체 공부 시간의 **60%**를 여기에 쓰는 게 과락 방어 + 총점 확보 양쪽에 유리합니다.</Highlight> 2024 개편 이후 기출은 객관식으로 연습해야 실전 감각이 맞아요.
