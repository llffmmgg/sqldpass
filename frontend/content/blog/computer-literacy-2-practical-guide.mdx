## 필기만 보고 실기 보러 가면 당황합니다

컴활 2급 필기 합격률은 60–70%대인데, 실기는 **30–40%대**로 떨어져요. 필기는 용어 객관식이라 기출만 돌려도 합격하는데, 실기는 **실제 엑셀 파일에서 결과물을 만들어야** 하기 때문이에요.

"필기 붙었으니까 실기도 금방 되겠지"라고 생각하면 시험장에서 함수 오타 하나로 감점 맞고 나옵니다. 실기는 실기대로 따로 준비하는 게 맞아요.

## 시험 구성 — 뭐가 나오나요?

| 항목 | 내용 |
|------|------|
| 구성 | 스프레드시트 실무(엑셀) 1과목 |
| 작업 영역 | 기본작업, 계산작업, 분석작업, 기타작업 |
| 시간 | 40분 |
| 합격 | **100점 만점에 70점 이상** |

1급처럼 액세스(DB) 과목이 없어서 **엑셀 하나만** 준비하면 돼요. 1급과 비교하면 확실히 가볍습니다.

**작업 영역별 배점 체감:**
- 기본작업(정렬·조건부 서식·고급 필터): 20점
- 계산작업(**함수 5–6문제**): 40점
- 분석작업(피벗·부분합): 20점
- 기타작업(매크로·차트): 20점

<Highlight color="amber">계산작업이 40점이라 함수 한두 개 놓치면 바로 탈락권이에요. 여기가 승부처입니다.</Highlight>

## 빈출 함수 TOP — 이것만 외워도 반은 됩니다

실기에서 진짜 자주 나오는 함수만 모았어요. 개편 이후 추가된 `IFS`, `SWITCH`도 포함입니다.

| 함수 | 용도 | 체감 출제 빈도 |
|------|------|--------------|
| **IF / IFS** | 조건 분기 | 매회 출제 |
| **VLOOKUP / HLOOKUP** | 값 찾기 | 매회 출제 |
| **SUMIF / COUNTIF** | 조건부 합계·개수 | 매회 출제 |
| **AVERAGEIF** | 조건부 평균 | 자주 |
| **LEFT / RIGHT / MID** | 문자열 자르기 | 자주 |
| **LARGE / SMALL / RANK** | 순위·최대·최소 | 자주 |
| **ROUND / INT / MOD** | 숫자 처리 | 가끔 |
| **TODAY / NOW / YEAR** | 날짜 함수 | 가끔 |
| **AND / OR / NOT** | 논리 결합 | IF와 함께 |
| **SWITCH** | 다중 분기 | 개편 이후 신설 |

**공부 순서 추천:**
1. `IF` 중첩 연습 (조건 2–3개)
2. `VLOOKUP` — 4번째 인수(정확 일치 0/유사 1)부터 헷갈림
3. `SUMIF`, `COUNTIF`, `AVERAGEIF` 묶음으로
4. `LARGE`, `SMALL`, `RANK.EQ` 한꺼번에
5. 나머지는 기출에서 나온 패턴만

---

## VLOOKUP에서 가장 많이 감점돼요

실기 채점에서 제일 많이 틀리는 함수가 VLOOKUP이에요.

**자주 하는 실수:**
- **절대참조를 안 걸어서 범위가 바뀜** — `$A$2:$C$10`로 고정해야 해요
- 마지막 인수를 생략해서 **유사 일치(1)**로 들어감 → 엉뚱한 값 반환
- 찾을 값이 테이블의 **가장 왼쪽 열**에 있어야 한다는 조건을 놓침

<Highlight color="red">VLOOKUP 네 번째 인수를 생략하면 1(유사 일치)로 동작합니다. 정확하게 찾고 싶으면 반드시 `0` 또는 `FALSE`를 명시하세요.</Highlight>

## 차트 — 오탈자 하나로 감점

차트 문제는 함수보다 쉬워 보이는데 **감점 포인트가 많아요**. 주로 잡는 건 이런 거예요.

- 차트 종류 정확히 (꺾은선 vs 영역형)
- **축 제목, 차트 제목 텍스트 그대로** 입력
- 데이터 레이블 위치 (바깥쪽 끝, 안쪽 끝)
- 보조 축 사용 여부
- 범례 위치

문제에 "차트 제목은 '월별 매출'"이라고 쓰여 있으면 **따옴표 빼고** 정확히 `월별 매출`로 입력해야 해요. 한 글자라도 다르면 감점입니다.

## 매크로 — 단순 반복 작업

매크로 문제는 어려워 보이는데 실제로는 "기록 버튼 → 정해진 순서대로 클릭 → 정지" 수준이에요. VBA 코드를 직접 짜는 건 거의 안 나옵니다.

**자주 나오는 매크로 패턴:**
- 특정 범위에 **조건부 서식 적용**하는 매크로
- **정렬** 매크로
- **차트 생성** 매크로

문제에 적힌 순서를 **그대로** 따라가면 되고, 매크로 이름과 버튼 위치만 조건대로 맞추면 됩니다.

## 2주 실기 학습 플랜

하루 2시간 기준, 필기는 붙었다는 전제예요.

| 기간 | 할 일 |
|------|------|
| **1–3일** | 기출 1회 풀어보며 유형 파악. 어떤 함수가 나오는지 체크 |
| **4–8일** | 빈출 함수 10개 집중 연습 (특히 IF 중첩, VLOOKUP) |
| **9–11일** | 차트·피벗·부분합 반복 연습 |
| **12–13일** | 매크로 + 기타작업 마무리 |
| **14일** | 최근 기출 1회 풀면서 시간 배분 체크 |

<Highlight color="green">실기는 문제집보다 기출 파일을 직접 다운받아서 풀어보는 게 훨씬 효율적이에요. 종이로 보는 것과 엑셀에서 치는 건 완전히 다릅니다.</Highlight>

## 시험 당일 주의

- 문제를 **끝까지 읽고** 시작하세요. 1문제가 여러 조건을 담고 있는 경우가 많아요
- 함수 입력 후 **Enter 누르고 결과 확인** — 그냥 넘기지 말기
- 저장은 **수시로** (Ctrl+S) — 시험장 PC 멈추면 되돌릴 수 없어요
- 계산작업 40분 중 **20분 안에 끝내는 것** 목표
- 마지막 5분은 **저장 확인 + 결과 재검토**
