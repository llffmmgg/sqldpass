## 결론 먼저

컴활 1급 필기 60문항 중 **40% 이상이 용어 매칭 단답 문제**예요. 이 글은 3과목별로 **매회 반복되는 용어**만 추려서 "용어 → 한 줄 정의" 형태로 정리했습니다. 시험 직전 30분 훑기용으로 썼을 때 가장 효율이 좋아요.

## 왜 용어 암기가 점수 결정 요인인가

컴활 1급 필기는 60문항 / 60분 / 과목당 20문항. 한 문제당 1분인데, **용어 단답 문제는 10–15초**면 답이 나와요. 빠르게 끝낸 단답 문제 수만큼 함수 계산·SQL 추적 같은 **시간 드는 문제**에 여유가 생깁니다.

또 과목별 40점 과락 기준이 있어, 각 과목에서 8문제는 확보해야 해요. 단답 용어 문제만 정확히 맞춰도 각 과목 과락은 거의 방어 가능합니다.

## 1과목: 컴퓨터 일반 빈출 용어

### 운영체제

| 용어 | 정의 |
|------|------|
| **프로세스** | 실행 중인 프로그램 (메모리 할당) |
| **스레드** | 프로세스 내 실행 단위 (자원 공유) |
| **문맥 교환 (Context Switch)** | CPU가 프로세스 전환 시 상태 저장·복원 |
| **페이징** | 메모리를 고정 크기 단위(페이지)로 분할 |
| **세그멘테이션** | 메모리를 가변 크기 단위(세그먼트)로 분할 |
| **스와핑** | 메모리 부족 시 일부를 디스크로 이동 |
| **교착상태(Deadlock)** | 프로세스가 서로 자원 대기로 멈춤 |

**교착상태 4조건**: 상호배제 / 점유와 대기 / 비선점 / 순환 대기 (4가지 **모두** 성립 시 발생)

### 네트워크

| 용어 | 정의 |
|------|------|
| **TCP** | 연결형, 신뢰성 보장, 3-way handshake |
| **UDP** | 비연결형, 빠르지만 신뢰성 없음 |
| **IP** | 패킷 단위 전송, 주소 지정 |
| **HTTP** | 웹 페이지 전송 (80번) |
| **FTP** | 파일 전송 (20, 21번) |
| **SMTP** | 메일 발송 (25번) |
| **POP3 / IMAP** | 메일 수신 |
| **DNS** | 도메인 ↔ IP 변환 |
| **DHCP** | IP 자동 할당 |

**OSI 7계층**: 응용 → 표현 → 세션 → 전송 → 네트워크 → 데이터링크 → 물리 (암기주문: **응표세전네데물**)

### 정보 보안

| 용어 | 정의 |
|------|------|
| **방화벽** | 외부↔내부 접근 제어 |
| **IDS** | 침입 탐지 (감지만) |
| **IPS** | 침입 방지 (감지 + 차단) |
| **대칭키** | 암복호화 같은 키 (AES, DES) |
| **비대칭키** | 공개키·개인키 쌍 (RSA) |
| **해시** | 단방향, 복호화 불가 (SHA) |
| **VPN** | 가상 사설망 |

### 하드웨어

| 용어 | 정의 |
|------|------|
| **CPU = ALU + CU + 레지스터** | 연산+제어+임시저장 |
| **레지스터** | CPU 내부 초고속 메모리 |
| **캐시** | CPU와 RAM 속도 차이 보완 (SRAM) |
| **RAM** | 휘발성, 주기억 |
| **ROM** | 비휘발성, BIOS |
| **SSD** | 반도체 저장, HDD보다 빠름 |

**메모리 계층**: 레지스터 → 캐시(L1/L2) → RAM → SSD/HDD (빠름·작음·비쌈 → 느림·큼·쌈)

## 2과목: 스프레드시트 일반 빈출 용어

### 셀 참조

| 용어 | 표기 | 특징 |
|------|------|------|
| 상대참조 | `A1` | 복사 시 행·열 모두 이동 |
| 절대참조 | `$A$1` | 복사해도 고정 |
| 혼합참조 | `$A1` / `A$1` | `$` 붙은 쪽만 고정 |

### 함수 — 빈출 TOP

| 함수 | 용도 |
|------|------|
| `SUMIF` / `SUMIFS` | 조건부 합계 |
| `COUNTIF` / `COUNTIFS` | 조건부 개수 |
| `VLOOKUP` / `HLOOKUP` | 값 찾기 |
| `INDEX` / `MATCH` | 범위+위치 조회 |
| `IF` / `IFS` | 조건 분기 |
| `LEFT` / `MID` / `RIGHT` | 문자열 추출 |
| `ROUND` / `INT` / `MOD` | 숫자 처리 |
| `RANK` / `LARGE` / `SMALL` | 순위·최대·최소 |
| `DSUM` / `DCOUNT` / `DAVG` | DB 함수 (조건 범위 지정) |

### 표시 형식 코드

| 코드 | 의미 | 예시 |
|------|------|------|
| `#` | 유효 숫자만 | `###` → 123 |
| `0` | 자릿수 없으면 0 | `000` → 023 |
| `#,##0` | 천 단위 구분 | 1,234 |
| `yyyy-mm-dd` | 날짜 | 2026-04-20 |

### 기타

| 용어 | 정의 |
|------|------|
| **매크로** | 반복 작업 자동화 (VBA 기반) |
| **피벗 테이블** | 대량 데이터 요약 분석 (행·열·값·필터 4영역) |
| **조건부 서식** | 조건에 따라 서식 자동 적용 |
| **고급 필터** | 복잡한 조건으로 데이터 추출 |
| **부분합** | 정렬 후 그룹별 소계 표시 |

## 3과목: 데이터베이스 일반 빈출 용어

### DB 기본

| 용어 | 정의 |
|------|------|
| **스키마** | DB의 논리적 구조 정의 |
| **인스턴스** | 특정 시점의 실제 데이터 |
| **뷰 (View)** | 가상 테이블 (저장공간 없음) |
| **인덱스** | 검색 속도 향상용 자료구조 |
| **트랜잭션** | 하나의 논리적 작업 단위 |

**ACID**: Atomicity(원자성), Consistency(일관성), Isolation(독립성), Durability(영속성)

### 키

| 용어 | 정의 |
|------|------|
| **슈퍼키** | 유일성만 만족 |
| **후보키** | 유일성 + 최소성 |
| **기본키** | 후보키 중 대표 (NOT NULL) |
| **대체키** | 후보키 중 기본키가 아닌 것 |
| **외래키** | 다른 테이블의 기본키 참조 |

### 관계 대수

| 기호 | 이름 | SQL 대응 |
|------|------|---------|
| σ (시그마) | 셀렉션 | WHERE |
| π (파이) | 프로젝션 | SELECT |
| ⋈ (보우타이) | 조인 | JOIN |
| ÷ | 디비전 | 포함 관계 |
| ∪ | 합집합 | UNION |
| ∩ | 교집합 | INTERSECT |

### 정규화

- **1NF**: 원자값
- **2NF**: 부분 종속 제거
- **3NF**: 이행 종속 제거
- **BCNF**: 비후보키 결정자 제거

### SQL

| 명령 | 분류 |
|------|------|
| CREATE, ALTER, DROP | **DDL** (정의) |
| SELECT, INSERT, UPDATE, DELETE | **DML** (조작) |
| GRANT, REVOKE | **DCL** (제어) |
| COMMIT, ROLLBACK | **TCL** (트랜잭션) |

## 자주 하는 실수

**1. OSI 7계층 순서 착각**
응표세전네데물 (위→아래) 방향. TCP/IP는 4계층(링크·인터넷·전송·응용).

**2. IDS와 IPS 혼동**
**IDS = 탐지(Detection)**, **IPS = 방지(Prevention)**. 약어 의미로 구분.

**3. 슈퍼키 vs 후보키 차이**
둘 다 유일성은 있음. **최소성 유무**가 차이.

**4. DDL/DML/DCL 분류 오류**
CREATE = DDL (정의), SELECT = DML (조작), GRANT = DCL (제어). 헷갈리면 머리글자 의미로.

**5. 절대참조 `$` 위치 실수**
`$A1` = 열만 고정, `A$1` = 행만 고정. 양쪽 다 고정은 `$A$1`.

## 시험장에서의 접근법

1. **용어 단답부터 빠르게** — 10–15초 안에 결정
2. **함수 문제는 보기 중 확실한 오답 먼저 소거**
3. **관계대수 기호 → SQL 변환**으로 이해
4. **과목별 시간 배분**: 각 20분 이내, 총 60분
