## 결론 먼저

안녕하세요. 문어입니다 🐙

**CBT 모의고사** 는 *Computer-Based Test* 의 약자로, 실제 자격증 시험과 같은 **컴퓨터 화면 환경에서 치르는 모의 시험**이에요. 종이 답안지가 아니라 모니터로 문제를 보고 마우스/키보드로 답안을 입력하죠. 2020년 이후 한국의 대부분 국가기술자격·민간 자격증이 CBT 로 전환됐고, 지금은 **모의고사도 CBT 환경에서 연습해야 실전 감각이 잡혀요**.

이 글에선 CBT 모의고사가 PBT(종이 시험)와 어떻게 다른지, 2026년 현재 어떤 자격증이 CBT 로 운영되는지, 그리고 CBT 모의고사를 잘 활용하는 방법을 정리했어요.

---

## CBT vs PBT — 한 눈 비교

| 항목 | CBT (Computer-Based Test) | PBT (Paper-Based Test) |
|------|---------------------------|-----------------------|
| 문제 제시 | 모니터 | 종이 책자 |
| 답안 작성 | 마우스 클릭 / 키보드 | OMR 카드 표기 |
| 즉시 채점 | <Highlight color="green">가능</Highlight> (일부 자격증) | 불가, 공식 발표까지 대기 |
| 문제 풀이 순서 | 자유 이동 | 자유 이동 |
| 시간 관리 | 화면 타이머 자동 | 본인이 시계 관리 |
| 오답 체크 | 표시 기능 제공 | 수기로 | 
| 시험장 | 전국 CBT 지정 센터 | 각 시도 교육기관 |

더 자세한 차이는 [CBT vs PBT 완벽 비교](/blog/cbt-vs-pbt) 글에서 다뤘어요.

CBT 의 가장 큰 장점은 **시험 직후 결과 확인**이에요. 일부 자격증(컴활 실기 등)은 시험장을 나서기 전에 합불이 뜹니다. 반대로 단점은 **화면 피로도**와 **스크롤 이동** 때문에 종이로 보던 사람은 적응 시간이 필요해요. 이 적응을 위해서라도 **CBT 모의고사는 필수**입니다.

## 2026년 CBT 도입 자격증 목록

한국에서 CBT 로 치르는 주요 자격증은 다음과 같아요.

### IT · 데이터 계열
- **SQLD** — 한국데이터산업진흥원, 완전 CBT. [SQLD 기출 풀러 가기](/learn/sqld)
- **정보처리기사 필기** — 한국산업인력공단, CBT. [정처기 필기 기출](/learn/engineer-written)
- **정보처리기사 실기** — CBT 로 전환 중(2020년 이후). [정처기 실기 기출](/learn/engineer)
- **ADsP** — 한국데이터산업진흥원, 2024 개편 후 CBT. [ADsP 기출](/learn/adsp)
- **빅데이터분석기사** — CBT
- **리눅스마스터** — CBT

### 사무 · 오피스 계열
- **컴퓨터활용능력 1급 필기/실기** — 대한상공회의소, 완전 CBT. [컴활 1급 기출](/learn/computer-literacy-1)
- **컴퓨터활용능력 2급 필기/실기** — CBT. [컴활 2급 기출](/learn/computer-literacy-2)
- **ITQ** — 일부 CBT

### 금융 · 공기업
- 은행 FP, 증권/파생상품 투자자문인력 등 — 부분 CBT
- 일부 공무원 채용 직렬 — 2025년부터 CBT 확대 중

대부분 시험이 CBT 로 이동한 만큼, **"CBT 모의고사" 연습 없이는 종이로만 기출 풀고 시험장 가는 건 체감 난이도가 올라가요**.

## CBT 모의고사를 잘 활용하는 3가지 방법

### 1. <Highlight color="green">실전 타이머와 함께 풀기</Highlight>

CBT 환경의 핵심은 **화면 타이머의 압박감**이에요. 종이로 풀 땐 손목시계를 안 보면 되지만 CBT 는 계속 시간이 눈에 들어와요. 이 긴장감에 적응하는 유일한 방법은 **모의고사도 타이머 켜고 풀기**입니다.

문어CBT 는 각 자격증 실제 시험 시간과 동일하게 타이머가 작동해요. SQLD 는 90분, 정처기 실기는 180분, ADsP 는 90분 등. [모의고사 시작하기](/mock-exams)

### 2. <Highlight color="amber">오답을 CBT 환경에서 재풀이</Highlight>

종이 오답노트에 틀린 문제를 적고 다시 푸는 방식은 **CBT 시험에선 효과가 약해요**. 실제 시험은 화면에서 답을 찍는 거니까, 복습도 화면에서 해야 감각이 유지됩니다.

문어CBT 의 [오답노트](/wrong-answers) 는 틀린 문제를 자동 누적해서 같은 CBT 환경에서 다시 풀 수 있게 해요. 화면·마우스·타이머 조합 그대로.

### 3. 문제 표시 기능 연습

CBT 시험장에서는 **"나중에 다시 볼 문제 표시"** 기능이 있어요. 이걸 활용하면 어려운 문제를 건너뛰고 쉬운 문제부터 빠르게 확보할 수 있습니다. 하지만 평소 연습을 종이로만 했으면 이 기능을 쓸 생각을 못 해요. 모의고사 단계부터 **표시·건너뛰기·빠른 이동** 을 연습하세요.

| 기능 | 설명 |
|------|------|
| **실전 CBT 환경** | 실제 시험과 동일한 화면·타이머·문제 이동 |
| **매번 새 문제** | AI 가 기출 변형 문제를 매번 새로 구성 |
| **오답 자동 복습** | 틀린 문제를 자동 누적, 같은 CBT 환경에서 재풀이 |

| **지원 자격증** | SQLD · 정처기 필기/실기 · 컴활 1/2급 · ADsP |
| **완전 무료** | 광고 일부 있지만 모든 기능 무료 |

무엇보다 이름이 **문어CBT**인 건, 다리가 8개라 여러 자격증을 동시에 잡을 수 있다는 문어의 친근함 때문이에요 🐙.

## 정리

CBT 모의고사는 이제 "선택이 아닌 필수" 예요. 실제 자격증 시험 대부분이 CBT 로 바뀌었고, 종이 기출만 보고 가면 시험장 적응에만 시간을 쓰게 됩니다. 타이머 압박, 화면 피로, 문제 표시 기능 — 이 3가지를 모의고사 단계부터 같이 연습하세요.
