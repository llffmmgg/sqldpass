## 결론 먼저

취준·학업·알바와 자격증 공부를 병행할 때 가장 효과적인 건 **"주 단위 고정 시간 블록"** 이에요. 매일 불규칙하게 하면 번아웃 빠르고, 완벽히 매일 하려다 하루 건너뛰면 자책으로 중단합니다. **주 3–4회 × 1.5–2시간** 블록이 현실적이고 지속 가능한 리듬입니다.

---

## 왜 병행이 어려운가

### 의지 부족 아님 — 체력·집중력 문제

하루 8시간 일한 후 2시간 공부하는 건 **생물학적으로** 쉽지 않아요. 학교·알바로 소진된 정신력으로 암기·문제풀이를 하려면 효율이 평소의 30–50% 수준.

### 경쟁 우선순위

- 취업 공고 확인·자소서 수정
- 인턴·프로젝트 참여
- 학업·과제
- 건강·관계 유지

자격증이 이 모든 것 중 **몇 번째**가 되는가가 현실입니다.

### "매일 공부" 환상

SNS에 "매일 4시간 공부"하는 사람 보면 자책감. 하지만 대부분은 **주 3–4회 짧게**도 합격해요. 꾸준함이 빈도보다 중요.

---

## 시간 확보 3가지 전략

### 전략 1: 주 단위 고정 블록

**매주 같은 요일·같은 시간**에 공부. 예:

- 화·목 저녁 7–9시 (2시간)
- 토요일 오전 10–12시 (2시간)

총 주 6시간. 이 패턴을 **무조건 지킨다**는 원칙. 남는 시간은 공부 안 해도 됨.

### 전략 2: 출퇴근·등하교 시간

지하철·버스 30분 → 용어 암기용. 책 대신 **모바일 CBT**나 요약 카드가 현실적.

- 월–금 출근 30분 × 2회 = 주 5시간
- 암기·객관식 복습에 최적

### 전략 3: 주말 몰빵 (비추)

주중 0 + 주말 6–8시간. 단기적으로는 효과 있어 보이지만:

- 주중에 외운 것 잊어서 반복 손해
- 주말에 집중 안 되면 주 전체 날림
- 번아웃 빠름

**주 3–4회 분산**이 주말 몰빵보다 효율 높아요.

---

## 자격증별 현실적 병행 기간

"풀타임 기준 2주"라는 말에 속지 말고, **병행 상황**에선 2–3배 더 걸린다고 생각해야 해요.

| 자격증 | 풀타임 | 병행 (주 6시간) |
|-------|-------|----------------|
| SQLD | 2–3주 | **6–8주** |
| ADsP | 2–4주 | **6–10주** |
| 컴활 2급 (필기+실기) | 4–5주 | **10–12주** |
| 컴활 1급 (필기+실기) | 5–7주 | **3–5개월** |
| 정처기 필기 | 2–3주 | **5–8주** |
| 정처기 실기 | 2–3개월 | **4–6개월** |

병행 기간은 **여유 있게 잡되 꾸준한 루틴 유지**가 관건.

## 주 단위 플랜 예시

### 케이스 1: 직장인 (퇴근 후 공부)

| 요일 | 시간 | 내용 |
|------|------|------|
| 월 | 출퇴근 30분 | 용어 암기 |
| 화 | 19–21시 (2h) | 주요 내용 학습 |
| 수 | 출퇴근 30분 | 암기 반복 |
| 목 | 19–21시 (2h) | 기출 풀이 |
| 금 | 휴식 | - |
| 토 | 10–12시 (2h) | 약점 보강 |
| 일 | 휴식 | - |

총 주 **7–8시간**. 번아웃 방지 + 꾸준함.

### 케이스 2: 대학생 (과제·수업 병행)

| 요일 | 시간 | 내용 |
|------|------|------|
| 월 | 공강 1h | 기출 풀이 |
| 화 | 저녁 1.5h | 개념 학습 |
| 수 | 휴식 | - |
| 목 | 저녁 1.5h | 기출 풀이 |
| 금 | 휴식 | - |
| 토 | 오전 3h | 약점 보강 + 모의고사 |
| 일 | 휴식 (재충전) | - |

총 주 **7시간**.

### 케이스 3: 취준생 (자소서·면접 병행)

자소서 쓰고 면접 준비하는 시기엔 자격증은 **보조**. 매일 1시간 이상 쓰지 않기.

| 요일 | 자격증 | 취업 준비 |
|------|--------|----------|
| 월 | 30분 (출퇴근) | 자소서 1시간 |
| 화 | 1시간 | 면접 연습 |
| 수 | 30분 | 자소서 수정 |
| 목 | 1시간 | 기업 정보 조사 |
| 금 | 휴식 | 휴식 |
| 토 | 2시간 | 자소서 마무리 |
| 일 | 1시간 | 면접 준비 |

총 자격증 주 **6시간**, 취업 준비 병행.

## 번아웃 방지 5원칙

### 1. 휴식 요일 지키기

주 1–2일은 **완전히 공부 안 함**. 이게 장기 지속의 핵심.

### 2. 작은 단위로 쪼개기

"오늘 함수 10개 외우기"보다 **"오늘 함수 3개"**. 성공 경험 누적.

### 3. 진도보다 이해

급해서 진도 빼기만 하면 시험장에서 무너져요. 이해 안 되면 한 주제에 며칠 투자.

### 4. 모의고사로 동기부여

점수가 오르는 걸 **수치로 확인**하면 지속력 생김. 주 1회 모의고사 기록.

### 5. 완벽주의 버리기

"하루 3시간 못 하면 그날 포기" → 번아웃. **30분이라도 하면 OK**.

## 효율 높이는 습관

### 뽀모도로 기법 (25분 + 5분 휴식)

25분 집중 + 5분 휴식 반복. 2시간 공부하면 뽀모도로 4세트.

### 하루 첫 30분은 복습

어제 공부한 것부터 다시. 반복이 기억 정착의 열쇠.

### 스마트폰 격리

공부 시간 동안 폰은 다른 방에. 알림 꺼두기.

### 같은 장소에서 공부

환경이 고정되면 집중 시작까지 시간 단축.

### 기록하기

공부 시간·내용·모의고사 점수 기록. **뭘 얼마나 했는지** 눈으로 확인.

---

## 포기 유혹 올 때

### "시간이 너무 없어요"

→ 주 6시간도 어렵다면 **자격증 1개만** 선택. 여러 개 동시 X.

### "진도가 안 나가요"

→ 이해 안 되는 주제는 일단 넘기고 **전체 훑기 먼저**. 나중에 돌아오면 이해 됨.

### "모의고사 점수가 안 올라요"

→ 오답 분석 방식을 바꿔보세요. "틀렸다"가 아니라 **"왜 틀렸나"**.

### "피곤해서 책 못 보겠어요"

→ **오늘은 모의고사 객관식만 풀기**. 체력 없는 날엔 저강도 학습으로.

## 자주 하는 실수

**1. "하루 3시간" 플랜**
일주일 못 가서 좌절. 주 6–10시간을 요일 분산하는 게 현실적.

**2. 시간 재지 않음**
뭘 얼마나 했는지 모르면 개선 불가. 최소 **총 공부 시간** 기록.

**3. 연속 학습**
쉬는 날 없으면 번아웃. 주 1일은 무조건 OFF.

**4. 여러 자격증 동시**
자격증 2개 이상 동시 준비는 비추. 1개씩 집중.

**5. 시험일부터 역산 안 함**
"언젠가 붙겠지"로 하면 안 됨. **시험일 정하고 역산**해서 주 단위 플랜.

---

## 시험장/준비 관점 접근법

1. **시험일 먼저 예약** — 역산 공부 시작
2. **주 단위 고정 블록** — 불규칙 금지
3. **주 1일 휴식** 필수
4. **모의고사 주 1회** — 진도 확인
5. **취업·학업 균형** — 자격증만 매달리기 X

<Highlight color="green">"매일 못 하는 날"이 정상이에요. 주 6시간만 지켜도 3개월이면 자격증 1개는 딸 수 있습니다.</Highlight>

## 정리

- 매일 공부 환상 버리고 **주 단위 블록** 관리
- 주 6–10시간 × 3–4회 분산 배분
- 주 1일은 완전 휴식
- 병행 기간은 풀타임의 **2–3배**로 넉넉히 잡기
- 여러 자격증 동시 금지 — 1개씩 집중
