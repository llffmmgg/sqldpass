## 결론 먼저

2026 컴활 2급은 **필기 40문항 / 40분**, **실기 Excel 2021** 기준입니다. 필기는 과목당 40점 이상 + 평균 60점, 실기는 100점 만점 70점 이상이면 합격. 이 글은 2024년부터 적용된 신(新)출제 기준(2024~2026)에 맞춰 컴퓨터 일반·스프레드시트 핵심만 추려둔 입문자용 개념 노트예요.

---

## 왜 2026에 다시 정리하나

컴활 2급은 2021년부터 정기검정이 폐지되고 **상시검정**으로만 운영돼요. 시험장·일정이 주 단위로 바뀌고, 2024년 개편으로 Excel 버전이 **2016 → 2021**로 올라가면서 `IFS`, `SWITCH`, `XLOOKUP` 같은 신규 함수가 실무 교재에 들어왔습니다. 이 글은 "2024~2026 출제 기준" 기준으로, 오래된 자료와 혼동하기 쉬운 포인트를 짚어가며 정리한 자료예요. 입문자가 가장 많이 헷갈리는 "어디까지 외워야 하나?" 질문에 답하는 게 목표입니다.

## 1과목 · 컴퓨터 일반

2026 출제 기준상 비중이 큰 영역은 **운영체제(Windows) → 인터넷/네트워크 → 멀티미디어 → 정보보안** 순서예요. 최근 기출에서 꾸준히 나오는 포인트를 정리했습니다.

### Windows 운영체제

- **작업 관리자**: `Ctrl + Shift + Esc` — 응답 없는 프로세스 강제 종료
- **휴지통**: USB/네트워크 드라이브에서 삭제한 파일은 휴지통을 거치지 않음
- **바로 가기 아이콘**: 확장자 `.lnk`, 원본 파일을 삭제하면 작동 안 함
- **파일 속성**: 읽기 전용 / 숨김 / 보관 세 가지
- **레지스트리**: 시스템/응용 프로그램 설정이 저장된 계층형 DB (`regedit`)

### 멀티미디어 (2024 이후 비중 증가)

| 구분 | 대표 포맷 | 특징 |
|------|---------|------|
| 이미지 | BMP / JPG / PNG / GIF | JPG는 손실 압축, PNG는 투명도 지원 |
| 오디오 | WAV / MP3 / MIDI | MIDI는 음표 정보만 저장, 용량 작음 |
| 비디오 | AVI / MP4 / MOV | MP4는 H.264/H.265 코덱 |
| 압축 | ZIP / RAR / 7Z | 무손실 압축 |

- **코덱(Codec)**: Coder + Decoder. 영상/음성 인코딩·디코딩 담당
- **스트리밍**: 전체 다운로드 없이 **받으면서 재생**
- **해상도 vs 화소**: 해상도는 가로×세로 픽셀 수, 1픽셀(화소)은 색 정보 단위

### 네트워크

- **IP 주소**: IPv4는 32비트(4바이트), IPv6는 128비트
- **서브넷 마스크**: 네트워크부와 호스트부 구분
- **DNS**: 도메인 ↔ IP 변환 (UDP 53번 포트)
- **HTTP/HTTPS**: 80 / 443 포트
- **FTP**: 파일 전송 (21번)
- **SMTP/POP3/IMAP**: 메일 송신(25) / 수신(110) / 서버 동기화(143)

### OSI 7계층 (암기 필수)

응용(7) → 표현(6) → 세션(5) → 전송(4) → 네트워크(3) → 데이터링크(2) → 물리(1)

| 계층 | 전송 단위 | 대표 장비 |
|------|---------|----------|
| 전송 | 세그먼트 | — |
| 네트워크 | 패킷 | 라우터 |
| 데이터링크 | 프레임 | 스위치·브리지 |
| 물리 | 비트 | 허브·리피터 |

### 정보 보안

2024 개편 이후 **보안 용어 단답형** 비중이 늘었어요. 아래 용어는 정의 한 줄씩 암기.

- **랜섬웨어**: 파일을 암호화한 뒤 복호화 대가로 금전 요구
- **피싱(Phishing)**: 정상 사이트를 위장해 개인정보 탈취
- **파밍(Pharming)**: DNS를 위조해 가짜 사이트로 유도
- **스미싱(Smishing)**: SMS + 피싱
- **방화벽**: 외부 → 내부 접근 제어 (내부 → 외부는 통과)
- **IDS / IPS**: 탐지만 / 탐지 + 차단
- **대칭키**: AES, DES (빠름, 키 공유 문제)
- **비대칭키**: RSA (공개키 + 개인키)

<Highlight color="red">"방화벽은 내부에서 외부로 나가는 공격을 막는다"는 **오답**이에요. 기본 방향은 "외부 → 내부" 접근 제어입니다.</Highlight>

## 2과목 · 스프레드시트 일반

2026에도 함수·셀 참조·표시 형식·피벗·매크로는 그대로 핵심이에요. 단, 2024 개편으로 **신규 함수(IFS, SWITCH)** 가 추가됐고, 실무에선 **XLOOKUP**도 언급되지만 2급 **필기**는 여전히 VLOOKUP 중심입니다.

### 필수 함수 11선 (2026 기준)

| 함수 | 기능 | 예시 |
|------|------|------|
| `SUM` | 합계 | `=SUM(A1:A10)` |
| `AVERAGE` | 평균 | `=AVERAGE(B1:B10)` |
| `IF` | 조건 분기 | `=IF(A1>=60,"합격","불합격")` |
| `IFS` | 다중 조건 (2024↑) | `=IFS(A1>=90,"수",A1>=80,"우",TRUE,"미")` |
| `VLOOKUP` | 세로 찾기 | `=VLOOKUP(찾을값, 범위, 열번호, 0)` |
| `HLOOKUP` | 가로 찾기 | `=HLOOKUP(A1, B1:F5, 3, 0)` |
| `COUNTIF` | 조건부 개수 | `=COUNTIF(A1:A10,">=60")` |
| `SUMIF` | 조건부 합계 | `=SUMIF(A1:A10,"남",B1:B10)` |
| `RANK.EQ` | 순위 | `=RANK.EQ(B2,$B$2:$B$10,0)` |
| `LEFT/MID/RIGHT` | 문자열 추출 | `=MID("20260425",5,2)` → `"04"` |
| `ROUND` | 반올림 | `=ROUND(3.456,2)` → `3.46` |

### 실전 수식 예시 4개

**1) 점수 → 등급 (중첩 IF 대신 IFS로)**

```excel
=IFS(B2>=90,"A", B2>=80,"B", B2>=70,"C", TRUE,"F")
```

**2) 부서별 평균 (조건부 평균)**

```excel
=AVERAGEIF(C2:C100,"영업팀",D2:D100)
```

**3) 이름 첫 글자 + 뒤 2자리 마스킹**

```excel
=LEFT(A2,1) & "**" & MID(A2,LEN(A2),1)
```

**4) 사번으로 이름 찾기 (VLOOKUP 고정참조)**

```excel
=VLOOKUP(E2, $A$2:$C$100, 3, 0)
```

### 셀 참조 3종 (헷갈리는 1순위)

- `A1` — **상대참조**, 복사하면 같이 이동
- `$A$1` — **절대참조**, 어디로 복사해도 고정
- `$A1` / `A$1` — **혼합참조**, 열만 / 행만 고정

<Highlight color="yellow">수식을 아래로 끌어 복사할 때 범위가 밀려서 오답이 나오는 경우는 대부분 `$` 를 빠뜨린 상대참조 실수예요. VLOOKUP의 **두 번째 인수(범위)**는 99% 절대참조.</Highlight>

### 표시 형식 코드

| 코드 | 의미 | 예시 |
|------|------|------|
| `#` | 유효 숫자만 | `###` → 123 |
| `0` | 자리수 없으면 0 | `0000` → 0023 |
| `#,##0` | 천 단위 쉼표 | 1,234 |
| `0.0%` | 백분율 소수 1자리 | 0.125 → 12.5% |
| `yyyy-mm-dd` | 날짜 | 2026-04-25 |
| `@` | 문자 자리 | 그대로 표시 |

### 차트·피벗테이블·매크로

- **차트**: 원형(비율) / 꺾은선(추이) / 막대(비교) / 분산형(상관)
- **피벗테이블 4영역**: 행 · 열 · 값 · 필터. 원본 데이터는 건드리지 않음
- **매크로 파일 확장자**: `.xlsm` (일반 `.xlsx`는 매크로 저장 불가)
- **매크로 보안**: "모든 매크로 제외" / "디지털 서명된 매크로만" 등 4단계
- **2급 매크로 출제**: 지시대로 기록 → 도형 연결이 대부분. **상대 참조 매크로·VBA 코딩은 2급 실기 출제 범위 아님**

## 초보가 자주 하는 실수

- **"필기 붙으면 실기 여유로움"** — 아닙니다. 2급은 실기 탈락률이 더 높아요. 필기 합격 후 2년 내 실기 합격 필수.
- **함수 이름만 외우고 인수 순서 안 외움** — VLOOKUP 4번째 인수(`0` vs `1`)를 틀리면 정답이 통째로 달라짐. `0`(FALSE)은 정확히 일치, `1`(TRUE)은 근사치.
- **절대참조 `$` 위치 반대로 입력** — `$A1`과 `A$1`을 바꿔 쓰면 표가 완전히 깨져요.
- **피벗테이블이 원본을 수정한다고 오해** — 피벗은 **요약 뷰**, 원본은 그대로.
- **매크로 파일을 `.xlsx`로 저장** — 매크로가 통째로 사라집니다. 반드시 `.xlsm`.

## 시험장 접근법

- **1과목(컴퓨터 일반) 먼저** — 단답 기반이라 10~20초에 끝남. 여기서 시간을 벌어 2과목에 몰아쓰기.
- **함수 문제는 보기 먼저** — 보기에 결과값이 적혀있으면 머릿속 실행 → 매칭이 수식 해석보다 빠름.
- **30초 룰** — 30초 안에 답 안 나오면 체크 후 넘기고 마지막에 돌아오기.
- **평균 60점 + 과목별 40점** — 약한 과목 40점 커트라인부터 방어한 뒤 평균 점수 끌어올리기.
- **실기 연습은 반드시 시험장과 같은 Excel 2021 환경에서**. 2016·365는 일부 메뉴 위치가 다름.
