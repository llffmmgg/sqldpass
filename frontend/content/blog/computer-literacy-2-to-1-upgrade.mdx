## 결론 먼저

컴활 2급을 딴 상태라면 **1급 필기 준비 기간을 절반으로 단축**할 수 있어요. 2급의 2과목(컴퓨터 일반·스프레드시트)이 1급에도 그대로 있고, **1급은 데이터베이스(3과목)와 실기 액세스**만 새로 학습하면 됩니다. 이 글은 2급에서 1급으로 넘어갈 때 재활용할 부분과 새로 공부할 부분을 정리해요.

---

## 왜 2급 → 1급 순서가 유리한가

### 난이도 격차를 줄여서 갈 수 있음

컴활 1급을 처음부터 도전하면 3과목(DB)에서 SQL·정규화·키 개념이 낯설어 시간이 많이 걸려요. 2급을 먼저 따면:

- 시험 경험치 누적 (CBT 환경, 시간 배분 감각)
- 엑셀 함수 10여 개 이미 숙달
- 1과목 용어(OSI·TCP·보안)도 중복

그래서 1급 준비 기간을 단축할 수 있어요.

### 취업·이력서에 중간 성과 확보

2급만 따도 **사무직·행정직** 이력서에 쓸 수 있어요. 1급 준비 중에도 2급 자격증은 살아있으니까 취업 기회 놓치지 않음.

---

## 2급에서 1급으로 — 겹치는 부분

### 1과목: 컴퓨터 일반 (거의 동일)

2급 내용이 1급에도 대부분 포함. 추가되는 것:

- 멀티미디어 세부 (압축·코덱 등)
- 고급 하드웨어 (RAID, 가상화)
- 네트워크 심화 (서브넷, VLAN)

<Highlight color="green">1과목은 2급 공부한 걸 **복습 수준**으로 다시 보고, 추가 주제 3–4개만 잡으면 돼요.</Highlight>

### 2과목: 스프레드시트 (1급은 심화)

2급에서 배운 함수는 그대로 씀. 1급에서 **추가 학습** 필요한 함수:

- `INDEX + MATCH` (VLOOKUP의 상위 버전)
- `SUMIFS` / `COUNTIFS` / `AVERAGEIFS` (복수 조건)
- `DSUM` / `DCOUNT` / `DAVG` (데이터베이스 함수)
- `SUMPRODUCT` (배열 계산)
- **매크로·VBA** 기초

2급의 IF·VLOOKUP·SUMIF·COUNTIF 가 1급에선 **심화·조합**으로 나옴.

---

## 1급에서 새로 공부할 부분

### 3과목: 데이터베이스 일반 (완전 신규)

2급에는 없던 과목. 새로 공부해야 함.

| 주제 | 내용 |
|------|------|
| DB 기본 | 스키마, 뷰, 인덱스, 트랜잭션 |
| ACID | 원자성·일관성·독립성·영속성 |
| 키 | 기본·외래·후보·대체·슈퍼키 |
| 정규화 | 1NF ~ BCNF |
| SQL | SELECT/JOIN/GROUP BY/HAVING |
| 관계대수 | σ·π·⋈·÷ 기호 |

**팁**: SQLD 준비한 적 있으면 거의 공짜. 없어도 1–2주면 기본 개념 잡힘.

### 실기: 액세스 추가

2급 실기는 엑셀만. 1급 실기는 **엑셀(60점) + 액세스(40점)** 구조.

액세스는 대부분 처음 써보는 프로그램이라 **2–3주 별도 학습** 필요.

- 테이블 설계
- 쿼리 (SQL 기반)
- 폼·보고서 제작

## 1급 준비 기간 — 2급 보유자 기준

| 항목 | 처음 도전 | 2급 보유자 |
|------|----------|-----------|
| 1급 필기 | 2–3주 | **1–2주** |
| 1급 실기 | 4–6주 | **3–4주** |

전체 1급 취득 기간이 **5–6주** 수준으로 단축됩니다. 처음부터 1급 도전하는 사람은 **7–10주** 걸리는 것 대비 현저히 빨라요.

---

## 2급 → 1급 연결 학습 플랜 (5주 기준)

### 1주차: 1급 필기 1과목·2과목 복습

- 2급과 겹치는 부분은 **빠르게 훑기만**
- 1급 추가 주제 (RAID·고급 함수)만 집중

### 2주차: 1급 필기 3과목 (DB) 집중

- SQL 기본 문법
- 정규화·키
- 기출 3회분 풀기

### 3주차: 1급 필기 시험 응시 + 실기 시작

- 필기 시험 일정 잡고 응시
- 합격 후 바로 실기 준비 시작

### 4주차: 실기 엑셀

- 함수 심화(INDEX+MATCH, SUMIFS, D함수)
- 매크로·VBA 기초

### 5–6주차: 실기 액세스

- 액세스 설치 → 테이블·쿼리 실습
- 기출 파일 반복

## 자주 하는 실수

**1. 2급 합격 직후 쉬었다가 1급 시작**
필기 지식이 흐려져 1과목·2과목을 **처음 공부하는 것처럼** 다시 해야 함.

**2. 1급 실기의 액세스 난이도 과소평가**
"엑셀이니까 비슷하겠지" → 오산. **완전 다른 프로그램**.

**3. 1급 필기 합격 후 실기 미룸**
2년 유효기간 있지만 빨리 이어야 유리.

**4. 1급 3과목(DB)을 SQL 지식 없이 도전**
SQLD·정처기 경험 없으면 3과목만 2주 잡아야 함.

**5. 1급 실기 매크로·VBA 포기**
배점 꽤 크니까 기본 패턴 4가지(Range·Cells·For·If)는 익히고 가야.

## 2급과 1급 차이 — 취업 활용도

| 직무 | 2급 | 1급 |
|------|-----|-----|
| 사무·행정직 | ○ | ◎ |
| 공기업·전산직 | △ | ○ (가산점) |
| 비서·회계 | ○ | ◎ |
| 개발 직무 | 의미 적음 | 의미 적음 |

**공기업 가산점**은 1급에만 부여되는 경우가 대부분. 2급은 "기본 스펙" 정도이고, 1급은 "변별력 있는 스펙".

---

## 2급만 따고 1급 안 따도 되는 경우

- **사무직 일반 취업**: 2급으로 충분
- **시간·비용 제약 큼**: 2급만 확보하고 다른 자격증으로
- **IT 개발 직무 희망**: 컴활 자체가 가점 적음 → SQLD·정처기로 방향 전환

반대로 **공기업·공무원** 지원 예정이면 1급까지 권장. 가산점이 다릅니다.

---

## 시험장/준비 관점 접근법

1. **2급 합격 직후 2–4주 안 1급 시작** — 가장 효율적
2. **3과목(DB) 먼저 공략** — 유일하게 완전 신규 영역
3. **필기 합격 후 바로 실기** — 간격 최소화
4. **액세스는 설치부터** — 1일차에 무조건 설치하고 클릭해보기

<Highlight color="green">2급 보유자의 1급은 "부분 이어 붙이기" 공부라 시간 효율이 높아요. 6주 집중으로 충분히 합격권.</Highlight>

## 정리

- 1과목·2과목은 2급과 **대부분 겹침** → 빠르게 복습
- 3과목(DB)과 액세스는 **완전 신규**, 각 2–3주
- 전체 1급 취득 기간: 5–6주 (처음 도전 대비 절반 단축)
- 공기업·전산직 지원이면 1급까지, 일반 사무직은 2급으로도 OK
