## 결론 먼저

컴활 1급 필기 합격 직후 **2–4주 안에 실기 준비를 시작**하는 게 가장 효율적이에요. 필기 유효기간 2년을 믿고 미루면 외운 함수·용어가 희미해져서 실기 공부 시간이 훨씬 더 들어갑니다. 실기 합격률이 30%대로 낮은 만큼 타이밍이 중요해요.

---

## 왜 연결 타이밍이 중요한가

### 실기 합격률이 낮은 이유

| 구분 | 합격률 | 차이 |
|------|--------|------|
| 필기 | 50–60% | 이론 객관식 |
| 실기 | 30%대 | 실제 조작 |

필기는 이론만 외우면 되지만, 실기는 엑셀·액세스를 **직접 조작**해야 해요. 마우스 클릭 순서, 함수 괄호 위치, 절대참조 `$` 한 번 실수가 바로 감점.

### 2년 유효기간 — 안심하면 안 됨

필기 합격 후 실기 응시 기한은 **2년**. 넉넉해 보이지만 실제로는:

- 6개월 넘기면 필기 때 외운 함수·용어 기억 흐릿
- 1년 넘기면 다시 공부해야 하는 느낌
- 2년 꽉 채워 보는 사람 대부분이 탈락

필기에서 쌓은 **함수 지식이 실기에서 그대로 쓰이니까**, 가까운 시일에 이어 가는 게 시간 효율이 제일 좋아요.

---

## 실기 시작 타이밍 3가지 시나리오

### 시나리오 1: 합격 직후 바로 (1–2주 이내)

**추천 상황**:
- 현재 모멘텀이 있음
- 실기 시험장 자리가 확보된 상태
- 공부 피로가 아직 크지 않음

**장점**: 함수 지식이 생생함, 빠른 추가 합격
**단점**: 연속 학습 피로

### 시나리오 2: 2–4주 쉬고 시작 (권장)

**추천 상황**:
- 필기에서 번아웃 상태
- 다음 달 시험장 예약

**장점**: 회복 후 집중력 상승
**단점**: 없음 (가장 일반적 선택)

### 시나리오 3: 2–3개월 후 시작

**추천 상황**:
- 취준 등 다른 일로 바쁨
- 실기 시험 일정이 아직 여유 있음

**장점**: 다른 활동 병행 가능
**단점**: 필기 때 외운 **함수·용어 재학습** 필요 (1주 추가)

<Highlight color="green">대부분의 합격자는 **시나리오 2**를 선택해요. 2–3주 쉬고 4주 집중이 현실적으로 가장 안정적입니다.</Highlight>

## 필기-실기 연결 관계

### 함수 — 거의 그대로 이어짐

필기에서 개념 암기한 함수가 실기에선 **직접 타이핑** 대상.

| 필기 | 실기 |
|------|------|
| VLOOKUP 4번째 인수 의미 묻기 | VLOOKUP 직접 작성 |
| SUMIF vs SUMIFS 차이 묻기 | 조건 맞춰 SUMIFS 작성 |
| 절대참조 이론 | `$A$1` 실제 입력 |

필기에서 익힌 함수를 **실기에선 오타 없이 치는** 단계로 발전시키면 돼요.

### SQL — 액세스에 연결

필기 3과목의 SQL 이론이 실기 액세스에서 쿼리로 구현.

| 필기 | 실기 |
|------|------|
| SELECT/GROUP BY 결과 예측 | 액세스 쿼리로 직접 작성 |
| 정규화 이론 | 테이블 생성에 반영 |
| 키 종류 구분 | 기본키·외래키 지정 |

### 개념 — 직접 연결 적음

OSI 7계층, 진수 변환 같은 1과목 개념은 실기와 직접 연결 적음. 실기 공부 시에는 엑셀·액세스 실습에만 집중하면 됨.

---

## 실기 준비 기간 가이드

필기 끝난 직후 기준.

| 상태 | 엑셀 | 액세스 | 총 기간 |
|------|------|--------|--------|
| 엑셀 경험 있음 | 1–2주 | 2–3주 | **3–4주** |
| 엑셀 초보 | 2–3주 | 2–3주 | **4–5주** |
| 완전 비전공자 | 3주+ | 3주+ | **5–6주** |

**액세스가 엑셀보다 오래 걸리는 이유**: 엑셀은 일상에서 경험 있지만 액세스는 대부분 처음 써봐서.

## 실기 시작 직후 1주일 — 이렇게 하세요

### 1–2일차: 기출 1회 진단

- 실기 기출 1회분 다운로드 후 실제 조작해 풀어보기
- **엑셀·액세스 각각 몇 점 나오는지** 확인
- 어떤 함수·쿼리가 나오는지 유형 파악

### 3–5일차: 엑셀 함수 집중

- VLOOKUP · INDEX+MATCH · SUMIFS · D함수 반복
- 오타 없이 타이핑 하는 연습
- 절대참조 `$` 위치 감각 익히기

### 6–7일차: 액세스 설치 + 친숙해지기

- Microsoft 365 체험판 또는 교재 부록
- **테이블·쿼리·폼·보고서** 하나씩 직접 만들어보기
- SQL 보기와 디자인 보기 왕복 연습

---

## 2주차 이후 — 본격 기출 반복

### 2주차: 쿼리 집중

액세스 쿼리가 배점 가장 큼. 기출 쿼리 20개 정도를 **디자인 보기와 SQL 보기 둘 다**로 풀어보기.

### 3주차: 차트·피벗·매크로

엑셀의 분석작업·기타작업. 매크로 VBA 기본 패턴 4가지 복습.

### 4주차: 모의고사·시간 배분

실제 90분 시간 재고 풀기. 엑셀 50–55분, 액세스 35–40분 배분 감각 익히기.

---

## 자주 하는 실수

**1. "필기 붙었으니 실기는 금방"**
실기 합격률이 30%라는 걸 간과. 최소 3–4주는 필요.

**2. 기출 없이 교재만**
실기는 **직접 조작**이 전부. 교재 정독해도 엑셀 안 열면 실전에서 막혀요.

**3. 액세스 미리 안 만져봄**
처음 열어본 상태로 시험장 가면 탈락 확정 수준. 최소 1주는 액세스 실습.

**4. 필기 끝나자마자 1주 안에 실기 응시**
함수 지식이 살아있는 건 맞지만, 실기 특화 연습 없이는 함정에서 감점.

**5. 2년 꽉 채워 공부 재개**
필기 때 외운 거 70%는 잊어요. 시간 효율로는 최악.

---

## 시험장에서의 접근법 (실기)

필기와 실기는 시험장 분위기부터 다릅니다.

1. **저장 `Ctrl+S` 수시로** — 중간에 PC 멈추면 되돌릴 수 없음
2. **엑셀 먼저 50분, 액세스 35분** — 시간 배분 정해놓고 시작
3. **함수 입력 후 결과 확인** — Enter 누르고 값이 나오는지
4. **매크로는 `.xlsm` 저장** — 기본 `.xlsx`면 매크로 삭제
5. **마지막 5분 저장 재확인**

<Highlight color="red">실기에서 가장 치명적 실수는 "저장 안 하고 제출". 시간에 쫓기면 그냥 나가는 경우가 있는데 0점 처리됩니다.</Highlight>
