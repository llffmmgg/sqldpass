## 실기 합격률이 <Highlight color="red">6%</Highlight>입니다

컴활 1급은 2024년 개편 전까지는 "좀 어렵지만 할 만한" 시험이었습니다. 개편 이후 실기 합격률이 **한 자릿수**로 추락했어요. 사무 분야 국가기술자격증 중 가장 어려운 시험이 됐습니다.

## 2024년 개편 — 뭐가 달라졌나

| 항목 | 개편 전 | 개편 후 |
|------|---------|---------|
| 엑셀 함수 | SUM, IF 수준 | <Highlight color="red">배열 수식, INDEX+MATCH 복합</Highlight> |
| VBA/매크로 | 거의 안 나옴 | <Highlight color="red">정기적으로 출제</Highlight> |
| Access | 단순 조작 | <Highlight color="red">쿼리 작성 + 매크로 연결</Highlight> |
| 작업량 | 90분이면 여유 | 90분이 빠듯 |

특히 **VBA**가 추가된 게 가장 큰 변화입니다. 엑셀만 쓰던 사람한테 코드를 짜라는 거니까요.

## 필기 vs 실기, 왜 이렇게 차이가 나나요?

| | 필기 | 실기 |
|---|------|------|
| 유형 | 4지선다 | **실제 프로그램 조작** |
| 합격 기준 | 평균 60점 | <Highlight color="red">70점</Highlight> |
| 찍기 | 가능 (25% 확률) | 불가능 |
| 체감 난이도 | 기출 반복하면 됨 | 손이 기억해야 함 |

실기 합격 기준이 <Highlight color="red">70점</Highlight>인 게 치명적입니다. 60점도 아니고 70점이라 여유가 없어요.

## 한 자릿수를 뚫으려면

### <Highlight color="amber">엑셀 (45분)</Highlight> — 여기서 점수를 뽑아야 합니다

배점의 절반입니다. 엑셀에서 무너지면 합격이 안 돼요.

**반드시 익혀야 하는 것:**
- `VLOOKUP`, `INDEX+MATCH` — 거의 매회 출제
- `SUMPRODUCT`, `COUNTIFS` — 복합 조건 집계
- 배열 수식 (`Ctrl+Shift+Enter`)
- 피벗 테이블 만들기
- **VBA 기초** — 변수 선언, For문, If문

### <Highlight color="blue">Access (45분)</Highlight> — 안 써본 사람이 제일 많이 떨어지는 구간

평소에 Access를 써본 사람이 거의 없어요. 그래서 여기서 무너집니다.

**최소한 이건 해야 합니다:**
- 테이블 만들기 + 관계 설정
- 선택 쿼리, 매개변수 쿼리
- 폼/보고서 컨트롤 속성
- 매크로 연결

> Access는 **설치하고 직접 만져보는 것**이 전부입니다. 교재만 읽으면 시험장에서 손이 안 움직여요.

## 현실적인 조언

컴활 1급 실기는 벼락치기가 안 됩니다. 특히 개편 이후로는요.

- 최소 <Highlight color="amber">3–4주</Highlight> 실습 위주 준비
- 엑셀 + Access가 설치된 PC에서 **매일 1시간 이상** 연습
- 필기 합격하면 바로 실기 준비 시작 — 미루면 감을 잃습니다
