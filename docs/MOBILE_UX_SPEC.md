# Mobile UX — Cross-Platform Spec

Android(`mobile/`) 와 iOS(`ios/`) 의 5탭 정보 구조·동선·마이크로 UX 단일 진실 원천.
양 플랫폼 코드는 본 문서를 미러링한다. UI 폴리시(색·간격·아이콘 디테일)는 사용자가 별 스킬로 진행 — 본 문서는 **정보 구조·라우팅·동작 정책만**.

원본 결정: `~/.claude/plans/giggly-brewing-crystal.md` (Q1~Q17).

---

## 1. 탭 구성 (5탭, 좌→우)

| # | 탭 | 정체성 | 핵심 책임 |
|---|---|---|---|
| 1 | **홈** | 오늘 상태 + 다음 행동 추천 | 스트릭·이어풀기·자격증 6종 캐러셀 |
| 2 | **모의고사** | 실전 모드 시험 응시 | 자격증 칩 + 회차 카탈로그 + 최고점수 배지 + PASS+ 잠금 |
| 3 | **기출복원** | 연도·회차별 기출 풀이 | 자격증 칩 + 연도·회차 카드 + 채점 |
| 4 | **실전 문제** | 학습 모드 단일 채점 풀이 (10문 즉시 채점) | 자격증 칩 + 과목 카드 → SoloSolve 진입 |
| 5 | **내정보** | 통계 요약 + 학습 흐름 메뉴 + 계정·설정 | KPI 2x2 + 3그룹 메뉴 |

이전 5탭(홈/모의고사/오답노트/인사이트/마이) 에서 변경:
- **제거**: 오답노트, 인사이트 (둘 다 내정보 안 메뉴로 흡수)
- **신규**: 기출복원(모의고사 세그먼트에서 분리), 실전 문제(단일 채점 정문)

## 2. 각 탭 정보 위계 (위→아래)

### 2.1 홈
1. 인사말 헤더 (닉네임 + 짧은 멘트)
2. **스트릭 카드** — currentStreak + 오늘 풀이 여부 + 격려/위험 톤 분기
3. **이어풀기 추천 카드** — 마지막 자격증/과목/모드 기반 진입 (조건부: lastCert 있을 때만)
4. **자격증 6종 수평 캐러셀** — 각 카드 탭 시 소개 모달 시트

### 2.2 모의고사
1. 자격증 칩 (수평 스크롤, 필터)
2. 회차 카드 리스트 (회차명·문항수·시간·최고점수 배지·PASS+ 자물쇠)
3. 카드 탭 → 응시 약속 화면 → Runner

### 2.3 기출복원
1. 자격증 칩 (수평)
2. 연도 칩 또는 드롭다운
3. 연도·회차 카드 리스트
4. 카드 탭 → Runner (응시·채점)

### 2.4 실전 문제
1. 자격증 칩 (수평)
2. 과목 카드 그리드/리스트 (과목명·마지막 풀이 정답률 작게)
3. 카드 탭 → SoloSolveScreen (1문 즉시 채점)

### 2.5 내정보
1. 프로필 헤더 (닉네임 · 구독 상태 배지)
2. **KPI 2x2 아이콘 그리드**: 총 풀이 / 평균 정답률 / 최장 스트릭 / 합격 확률
3. **학습 섹션**: 오답노트 / 북마크 / 풀이 이력
4. **계정 섹션**: 닉네임 편집 / 결제·청구 / PASS+ 구독 관리
5. **설정·지원 섹션**: 테마 / 알림 / 피드백 / 공지·약관 / 로그아웃 / 회원 탈퇴

## 4. 양 플랫폼 동일 규칙 (14가지)

1. **탭 순서·이름**: 홈 / 모의고사 / 기출복원 / 실전 문제 / 내정보 — 좌→우 동일
2. **각 탭 정보 위계**: 위 § 2 그대로
3. **모달 시트 패턴**: iOS `.sheet(detents:)`, Android `ModalBottomSheet` — 결제·자격증 소개·신고
4. **풀이 정문**: 실전 문제 탭 (홈 이어풀기는 보조)
5. **잠긴 회차 클릭**: 페이월 시트 자동 오픈 (alert 금지)
6. **풀이 화면 햅틱·정답 시각**: `docs/SOLVE_SCREEN_SPEC.md` 매트릭스
7. **인증 게이트**: 앱 진입 즉시 로그인. 비로그인 사용 불가
8. **"내 자격증" 개념 없음**: 영구 설정 X. 매번 자격증 칩에서 선택
9. **빈 상태**: "안내 + in-place CTA" 단일 패턴. 마스코트·일러스트 없음
10. **풀이 모드 상단 라벨**:
    - 단일 채점: `랜덤 10문 / {자격증}`
    - 모의고사: `{회차명}`
    - 기출: `{연도}-{회차}회 기출`
11. **스트릭 위험 알림**: 홈 스트릭 카드 자체 색·문구 전환. 푸시 알림 없음
12. **검색·필터**: 칩 기반만. 텍스트 검색 도입 없음
13. **자격증 소개 시트**: 시험 정보 4종(시행처·문항수·시간·합격기준) + "PASS+ 소개" CTA 1개
14. **SessionComplete (단일 채점 결과)**: 중간 풍부도 — 점수 + 문제별 그리드 + 아코디언 해설

## 5. 자격증 6종 정적 데이터

웹 `frontend/src/lib/cert-tokens.ts` 가 라벨·색을 제공. 시험 정보(시행처·문항수·시간·합격기준) 는 양 플랫폼이 동일 const 로 보관:

- Android: `mobile/app/src/main/java/com/sqldpass/app/ui/home/CertCatalog.kt`
- iOS: `ios/Sqldpass/Core/Catalog/CertCatalog.swift`

| 자격증 | slug | 시행처 | 문항수 | 시간 | 합격기준 |
|---|---|---|---|---|---|
| SQLD | `sqld` | 한국데이터산업진흥원 | 50 | 90분 | 60점 이상 (과목별 40% 이상) |
| 정보처리기사 필기 | `engineer-written` | 한국산업인력공단 | 100 | 150분 | 평균 60점, 과목별 40점 이상 |
| 정보처리기사 실기 | `engineer-practical` | 한국산업인력공단 | 20 | 150분 | 60점 이상 |
| 컴퓨터활용능력 1급 필기 | `computer-literacy-1` | 대한상공회의소 | 60 | 60분 | 평균 60점, 과목별 40점 이상 |
| 컴퓨터활용능력 2급 필기 | `computer-literacy-2` | 대한상공회의소 | 40 | 40분 | 평균 60점, 과목별 40점 이상 |
| ADsP | `adsp` | 한국데이터산업진흥원 | 50 | 90분 | 60점 이상 (과목별 40% 이상) |
